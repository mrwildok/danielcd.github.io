Coworkers: Nichole Dzeka (Ux senior) Carolina Gutierrez (Design manager) from Accenture.

The challenge.

Create a digital enrollment process to avoid users going to the office during the authentication process.

It should allow users to purchase products using the application.

User pain points.

  • Authentication process too obsolete.

  • Log in to the applications take a lot of time.

  • The users get locked constantly.

  • The experiences are not consistent between services and digital products.

Business pain points.

  • There is no customer profiling; all users get the same options.

  • Users are not using the application because it does not allow them to access easier.

  • Call center are receiving all PQR from users.

  • There are a lot of users going to the office to the authentication process.

We worked together.

After applying design thinking in workshops with businesses, users, and technology, we got results that allowed us to design a solution centered on users and business needs.

Results:

  • Definition of design north stars.

  • Biometric technology.

  • Card password as authentication factor.

  • Security levels according to information type.

  • North star 1.
    Advisor.

    We know what users want and really need.

  • North star 2.
    Simple.

    Users can do anything more straightforward and fast

  • North star 3.
    Strong.

    A brand with strong voice.

  • North star 4.
    Digital.

    Self-management using one channel.

Biometric.
